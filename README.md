# ktracer
**kt**racer is a basic raytracer written entirely in Kotlin. 

It currently supports:
- Multiple objects (spheres, cylinders, planes, disks, triangles and convex polygons)
- Blinn-Phong shading
- Reflections
- Anti-aliasing (SSAA) 
- Depth of field
- JSON serialization for scene creation
